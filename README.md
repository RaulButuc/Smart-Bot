## SmartBot

You can have a chat with SmartBot over [here](https://hmfw8rsx.apps.lair.io/)

**Still under development**

## Known commands

* **hi**
* **set name**
* **how old are you**
* **ip**
* **help**
* **quit**

## Soon to be integrated
